## ðŸ“ Project Description

This project involved developing a custom WordPress website for a web design agency.

The workflow included:

* Converting Figma designs into pixel-perfect HTML, CSS, and JavaScript
* Building a fully responsive layout across desktop, tablet, and mobile devices
* Integrating the front-end into WordPress as a custom theme
* Implementing dynamic content management using Advanced Custom Fields (ACF)
* Populating the website with structured content
* Testing functionality, responsiveness, and performance

The website was built with a strong focus on visual accuracy, usability, and maintainability. The structure allows easy content updates without affecting layout integrity.

This project demonstrates the ability to translate creative design concepts into functional, scalable WordPress solutions aligned with modern web standards.
